
The files contained in this folder serve as examples of input data and corresponding output data generated when executing the following command.
For further details, please refer to the Supplementary Information file of our paper.

Example A

perl BED-Craft.pl Example_A_input_default.txt


Example B

perl BED-Craft.pl Example_B_input_genome_hg19_buffer_10000_chr_no.txt -genome hg19 -buffer 10000 -chr no


Example C

perl BED-Craft.pl Example_C_input_mm10.txt -genome mm10


Example D

perl BED-Craft.pl Example_D_input_genome_Danio_rerio_annotation.bed.txt -genome Danio_rerio_annotation.bed


Additional Example

perl BED-Craft.pl Example_ACMG_SF3.2_gene_list.txt
